Guide de création

Structure d'une Fiche

Identité Nom, prénom, âge, taille, poids. Année de naissance cohérente avec 2497. Affiliation Faction, rang actuel, Maison éventuelle, relation avec le Vornacht. Description physique Apparence, implants visibles, marques particulières, façon d'occuper l'espace. Description morale Personnalité, code moral, motivations profondes, contradictions internes. Histoire Parcours jusqu'au moment du RP. Comment ce personnage est arrivé là où il est. Compétences Ce que le personnage sait faire — avec ses limites explicites. Capacités spéciales Si Pestiféré : degré, nature, coût physiologique, fréquence d'usage. Œil de Dieu Statut : actif, piraté, silencié, extrait. Les implications se jouent en RP. Objectif personnel Ce que le personnage veut dans ce monde. Source d'interactions avec les autres.
